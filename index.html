Tentang Saya

Saya adalah mahasiswa aktif Program Studi D4 Kearsipan dan Sains Informasi di Universitas Terbuka, sekaligus seorang Admin Project yang berpengalaman dalam pengendalian dan monitoring progres proyek housing dan infrastruktur.

Saya terbiasa mengelola laporan harian, mingguan, dan bulanan sebagai dasar evaluasi manajemen, serta melakukan validasi dan sinkronisasi data antara sistem dan kondisi lapangan.

  • Monitoring dan controlling progres multi-proyek
  • Mengelola data proyek melalui sistem (ERP / Project System)
  • Automasi reporting menggunakan Google Apps Script
  • Analisis kendala proyek berbasis 4M

Dengan semangat belajar dan teknologi tepat guna sebagai prinsip kerja, saya siap memberikan kontribusi yang maksimal dalam setiap proyek.
